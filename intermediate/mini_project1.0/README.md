# 🚀 Day 7: Mini Project — Student Record System in C++

Welcome to **Day 7** of the C++ learning series! 🎯
Today, we apply everything we’ve learned so far by building a **real-world mini project**.

---

## 📌 Project Overview

In this project, we will create a **Student Record System** that can:

* Store student names and marks
* Display all student details
* Find the highest marks
* Count how many students passed

---

## 🧠 Concepts Used

This project combines:

* ✅ Data Types
* ✅ Conditions (`if-else`)
* ✅ Loops (`for`)
* ✅ Arrays
* ✅ Strings

---

## 💡 Problem Statement

Write a C++ program that:

1. Takes input for number of students
2. Stores:

   * Student names
   * Student marks
3. Displays all student details
4. Finds the highest marks
5. Counts number of students who passed (marks ≥ 40)
