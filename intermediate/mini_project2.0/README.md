# 🚀 Student Record Management System (C++ Project)

A simple and powerful **console-based application** built using core C++ concepts.
This project is designed to apply everything learned from basics to problem-solving.

---

## 📌 Project Overview

The **Student Record Management System** allows users to:

* Add student details
* Display all records
* Search for a student
* Sort student data
* Analyze performance

---

## 🎯 Concepts Covered

This project combines multiple C++ concepts:

* Variables & Data Types
* Arrays (1D)
* Strings
* Loops & Conditions
* Functions
* Searching Algorithms
* Sorting Algorithms

---

## ⚙️ Features

### 🔹 Add Student

Store student name and marks.

### 🔹 Display Students

View all stored records.

### 🔹 Search Student

* Linear search using name
* Efficient lookup logic

### 🔹 Sort Students

* Sort based on marks
* Uses Bubble Sort

### 🔹 Find Topper

* Identify student with highest marks

---

## 🧠 How It Works

* Data is stored using arrays
* Each student has:

  * Name (string)
  * Marks (integer)
* Menu-driven program for user interaction

---

## 🖥️ Sample Menu

```
1. Add Student
2. Display Students
3. Search Student
4. Sort Marks
5. Find Topper
6. Exit
```

## ▶️ How to Run

1. Compile the code:

```
g++ main.cpp -o app
```

2. Run the program:

```
./app
```
